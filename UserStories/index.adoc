*Kurrier Inc.* - First-Mile Parcel Pickup Platform

This documentation provides comprehensive user stories organized into epics for the Kurrier platform, a mobile app connecting consumers who need parcel shipping with gig-economy drivers who provide first-mile pickup to carrier hubs.

== Project Overview

**Project:** Kurrier Inc. Mobile Application +
**Purpose:** First-mile parcel pickup platform +
**Target Users:** Consumers (shippers) and Drivers (gig workers) +
**Key Differentiator:** On-demand pickup with real-time tracking and competitive carrier quotes

== Epic Structure

The Kurrier platform is organized into 10 major epics, split between Consumer and Driver experiences.
